# Product Landing Page - Original Trombones

This project is a professional product landing page created as part of the **freeCodeCamp Responsive Web Design Certification**. It showcases a business that sells handcrafted musical instruments, focusing on clean layout, responsive design, and user conversion.

## 🚀 Live Demo
You can view the live version of this project here:

## ✨ Features
* **Fully Responsive:** Optimized for desktop, tablet, and mobile screens.
* **Sticky Navigation:** A fixed header that allows easy access to different sections of the page.
* **Embedded Video:** An integrated "How It Works" section featuring a product showcase video.
* **Modern Layout:** Built using **CSS Flexbox** for a symmetrical and clean aesthetic.
* **Interactive Elements:** Includes hover effects on navigation links, product cards, and buttons.
* **Email Subscription:** A functional-ready form for newsletter sign-ups.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure for better accessibility and SEO.
* **CSS3:** Custom styling with Flexbox, media queries, and transitions.
* **Google Fonts:** "Lato" for professional typography.
* **FontAwesome:** High-quality icons for visual storytelling.

- [x] Navigation bar with an `id="nav-bar"`.

- [x] Embedded video with an `id="video"`.
- [x] A form with an `id="form"` and an email input with `id="email"`.
- [x] Responsive layout using CSS Media Queries.
