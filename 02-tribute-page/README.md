# Project: Tribute Page - Dr. Norman Borlaug

This is one of the initial certification projects from the **freeCodeCamp Responsive Web Design** curriculum. It demonstrates basic HTML structural skills and CSS styling for informational long-form content.

## 🛠 Features
- **Semantic HTML5:** Proper use of `<main>`, `<ul>`, and `<figure>` equivalent structures.
- **Responsive Images:** Implemented CSS rules to ensure images scale correctly across all device sizes.
- **Clean Typography:** Focused on readability and line spacing for historical data presentation.

## 📋 Technical Requirements
- [x] Includes a clear `#title` and `#main` element.
- [x] Responsive image implementation within an `#img-div`.
- [x] A detailed `#tribute-info` section using a chronological list.
- [x] External link to additional resources via `#tribute-link` with `target="_blank"`.

## 💡 Project Insights
This project served as an introduction to **document structure and accessibility**. The challenge was to present a large amount of text-based data (a timeline) in a way that remains readable and visually organized on both mobile and desktop screens.
