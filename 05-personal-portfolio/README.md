# Project 5: Personal Portfolio Webpage

This is the final certification project for the **freeCodeCamp Responsive Web Design** curriculum. It serves as a professional landing page to showcase my technical skills, completed projects, and contact information as a Web Developer.

## 🛠 Technologies Used
- **HTML5:** Semantic structure (nav, section, footer) and content hierarchy.
- **CSS3:** Flexbox and Grid systems, CSS Variables, and fluid typography using `clamp()`.
- **Responsive Design:** Mobile-first approach with Media Queries for cross-device compatibility.

## 📋 Completed User Stories
- [x] Fixed navigation bar at the top of the viewport.
- [x] Dedicated `welcome-section` and `projects` gallery.
- [x] Project tiles organized with a responsive Grid layout.
- [x] External link to my GitHub profile (`profile-link`).
- [x] High attention to visual hierarchy and accessibility standards.

## 💡 QA & Testing Notes
- The page has been tested across multiple viewports (Mobile, Tablet, Desktop) for responsiveness.
- Optimized CSS specificity to prevent selector conflicts.
